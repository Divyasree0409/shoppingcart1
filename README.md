# 🛒 Mini Shopping Cart

A simple Mini Shopping Cart web application built using HTML, CSS, and JavaScript.  
This project demonstrates basic e-commerce functionality like adding items to cart, viewing cart details, and calculating total price.

---

## 📌 Features
-  Add products to cart
-  Remove items from cart
-  Update quantity
- Auto total price calculation
-  Responsive design

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
